Boxplot Chart

The boxplot is used to represent the distribution of data using:
  • a box to represent the region that contains 50% of the datapoints,
  • the whiskers to represent the regions that contains 95% of the datapoints, and
  • a center line (in the box) that represents the median of the dataset.
For more information on boxplots you can visit wikipedia.

Chart Options

Dimensions

  • width
  • height
  • margins - css-like margins shorthand syntax (without units) or object with properties 'all', 'top', 'bottom', 'left' and/or 'right'. Margins consume the chart's width and height (inside margins).

Title properties

  • title
  • titlePosition
  • titleSize

Lifecycle

  • renderCallback - executed immediatly before render is called

Legend properties

  • legend (true|false)
  • legendPosition (right, top, bottom, left)
  • legendAlign (right, left, center, top, middle, bottom)
  • legendSize
  • legendDrawLine (true|false)
  • legendDrawMarker (true|false)
  • tooltipEnabled (true|false)

Boxplot properties

  • orientation (horizontal | vertical)
  • valuesVisible (true|false)
  • stacked (true|false)
  • axisBandSizeRatio - Ratio of occupied vs total size of category band. Default: 0.9 (90%)
  • boxSizeRatio - Ratio of box size vs category band size. Default: 0.3 (30%)
  • boxSizeMax - Maximum size of a box in pixels.

Line properties

  • linesVisible - Show or hide lines. Default: true
  • dotsVisible - Show or hide dots. Default: false
  • areasVisible - Show or hide areas. Default: false
  • nullInterpolationMode (none | linear) - Interpolation mode for lines. Default: 'none'

Axis properties

  • show{X,Y}Scale (true | false) - whether to show an axis. Default: true
  • plot2 (true | false) - whether to show the second axis. Default: false
  • xAxisPosition (bottom | top) - the position of the X axis. Default: bottom
  • yAxisPosition (left | right) - the position of the Y axis. Default: left
  • {x,y}AxisSize - the size of free dimension of an axis (X: height, Y: width). Default: 30
  • {x,y}AxisGrid (true | false) - show lines on all (major) ticks, except the first and the last. Default: false
  • {x,y}AxisEndLine (true | false) - show a line on the last (major) tick. Default: false
  • {x,y}AxisTitle - The axis title text. Default: null
  • {x,y}AxisTitleFont - The axis title font. Default: '10px sans-serif'
  • {x,y}AxisTitleSize - The size of the axis title. The size is the width when the axis is vertical and the height when it is horizontal. The title consumes space from the specified {x|y}AxisSize. Default: The required by the font height plus a proportional margin.
  • {x,y}AxisDomainRoundMode (none | nice | tick) - rounding mode of the domain bounds of a linear scale axis. Default: none
  • {x,y}AxisDesiredTickCount - desired number of (major) ticks of a linear scale axis. Default: 5
  • {x,y}AxisMinorTicks (true|false) - show minor ticks on a linear scale axis. Default: true

Extension Points

The following list serves as the basis for Boxplot extension points:
  • base_
  • title_
  • titleLabel_
  • {x,y}Axis_
  • {x,y}AxisRule_
  • {x,y}AxisGrid_
  • {x,y}AxisScale_
  • {x,y}AxisTicks_
  • {x,y}AxisMinorTicks_
  • {x,y}AxisLabel_
  • {x,y}AxisTitleLabel_
  • tooltip_
  • boxPanel_
  • box_
  • boxBar_
  • boxRuleMedian_
  • boxRuleMin_
  • boxRuleMax_
  • boxRuleWhisker_
  • chart_
Sample extension points:
  • boxBar_fillStyle: for setting the bars fill colors.
  • boxBar_strokeStyle and boxBar_lineWidth: for setting the stroke color and thickness.
  • boxRuleMedian_strokeStyle and boxRuleMedian_lineWidth: for setting the stroke color and thickness of the median line.
  • titleLabel_font
  • yAxisRule_strokeStyle
  • xAxisScale_dateTickPrecision
