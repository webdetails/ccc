j CCC Samples - Boxplot

Boxplot Chart

The boxplot is used to represent the distribution of data using:
  • a box to represent the region that contains 50% of the datapoints,
  • the whiskers to represent the regions that contains 95% of the datapoints, and
  • a center line (in the box) that represents the median of the dataset.
For more information on boxplots you can visit wikipedia.

Chart Options

Dimensions

  • width
  • height

Title properties

  • title
  • titlePosition
  • titleSize

Lifecycle

  • renderCallback - executed immediatly before render is called

Legend properties

  • legend (true|false)
  • legendPosition (right, top, bottom, left)
  • legendAlign (right, left, center, top, middle, bottom)
  • legendSize
  • legendDrawLine (true|false)
  • legendDrawMarker (true|false)
  • showTooltips (true|false)

Boxplot properties

  • orientation (horizontal | vertical)
  • showValues (true|false)
  • stacked (true|false)
  • panelSizeRatio - Ratio of the band occupied by the pane. Default: 0.5 (50%)
  • barSizeRatio - In multiple series, percentage of inner band occupied by bars. Default: 0.5 (50%)
  • maxBarSize - Maximum size of a bar in pixels.

Extension Points

The following list serves as the basis for Boxplot extension points:
  • base_
  • title_
  • titleLabel_
  • xAxis_
  • yAxis_
  • xAxisRule_
  • yAxisRule_
  • xAxisRuleGrid_
  • yAxisRuleGrid_
  • xAxisScale_
  • yAxisScale_
  • tooltip_
  • boxPanel_
  • box_
  • chart_
Sample extension points:
  • box_fillStyle: for setting the bars fill colors.
  • box_strokeStyle and bar_lineWidth: for setting the stroke color and thickness.
  • titleLabel_font
  • yAxisRule_strokeStyle
  • xAxisScale_dateTickPrecision
