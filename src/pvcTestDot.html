Dot Chart

A Dot chart consists of group of data points plotted on a simple scale. The Dot chart can be a very useful alternative to the horizontal bar chart, being the dot chart less cluttered and less redundant than that.

Chart Options

Dimensions

  • width
  • height

Title properties

  • title
  • titlePosition
  • titleSize

Lifecycle

  • renderCallback - executed immediatly before render is called

Legend properties

  • legend (true|false)
  • legendPosition (right, top, bottom, left)
  • legendAlign (right, left, center, top, middle, bottom)
  • legendSize
  • legendDrawLine (true|false)
  • legendDrawMarker (true|false)
  • showTooltips (true|false)

Data properties

  • timeSeries (true| false)
  • timeSeriesFormat. For example: "%Y-%m-%d"

Dot chart properties

  • orientation - horizontal or vertical. Default: vertical
  • showDots - Show or hide dots. Default: false
  • showValues - Show or hide line value. Default: false
  • panelSizeRatio - Ratio of the band occupied by the pane. Default: 0.5 (50%)

Extension Points

The following list serves as the basis for Dot chart extension points:
  • base_
  • title_
  • titleLabel_
  • xAxis_
  • yAxis_
  • xAxisRule_
  • yAxisRule_
  • xAxisScale_
  • yAxisScale_
  • xAxisRuleGrid_
  • yAxisRuleGrid_
  • tooltip_
  • chart_ - for the main chart Panel
  • dot_
Sample extension points:
  • base_fillStyle
  • titleLabel_font
  • dot_shape
