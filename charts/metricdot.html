Metrics charts are similar to line graphs in that they use horizontal and vertical axes to plot data points.

However, the base axis of metric charts is always continuous, instead of discrete, as in categorical charts. They both have a continuous orthogonal axis.

The big difference is that categorical charts group data points by category (base) value and show only one visual element per unique category value. Metric charts show a distinct visual element for each data point, whether their category value occurs in other data points or not.

Metrics charts can show you, visually, the strength, shape or the direction of the relationship between variables, and whether there are outliers in the data.
